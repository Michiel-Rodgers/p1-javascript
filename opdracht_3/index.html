Opdracht 2 JavaScript

Wat is het verschil met de script tag boven in te laden in de head tag t.o.v onder inladen aan het einde van je HTML tag?

Als je een script tag in de head tag laad, dan wordt de JavaScript code meteen geladen, voordat de rest van de HTML code geladen wordt.


Wat doet de attribuut “defer” in de script tag?

Defer zorgt ervoor dat de JavaScript pas word uitgevoerd als de rest van de HTML code geladen is.


Wat is het verschil tussen HTML en JavaScript?

HTML is de basis van de website, JavaScript is de code die de HTML pagina interactief maakt.


Wat is het verschil tussen innerHTML en innerText?

Met innerHTML kun je de volledige HTML inhoud van een element aanpassen en met innerText alleen de text in een element
